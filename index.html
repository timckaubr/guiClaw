Installation

1. Set Gateway Token

Paste your OpenClaw gateway token below. It will be saved securely in your browser's local storage.

2. Run System Setup

This script ensures you can run openclaw from any terminal. It will add its location to your shell configuration (~/.zshrc).

You can view the results in the "Troubleshooting" tab.

3. Reload guiClaw Proxy

If you make changes or the connection fails, restart the proxy server by running this command in your terminal:

cd ~/Desktop/guiClaw && ./start.sh
